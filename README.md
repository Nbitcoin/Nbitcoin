# Nbtc-Core
## Project Introduction
Focus on core protocol development for omnichain Bitcoin, with an emphasis on cross-chain asset interoperability, and provide infrastructure for the omnichain Bitcoin ecosystem.

## Technical Architecture
- Cross-Chain Protocol Layer: Supports asset interoperability between Bitcoin and multiple chains.
- Ecosystem Infrastructure: Provides node services, toolkits, and other underlying supports.

## Quick Start
### Environment Requirements
- Operating System: Linux / macOS / Windows
- Dependencies: Docker (no specific version required, latest version recommended)

### Startup Steps
1. Prepare the project files in the local directory
2. Enter the project root directory via the command line
3. Start the service: `docker compose up -d`
